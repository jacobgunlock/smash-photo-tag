Super Smash Bros Where's Waldo

- A tiny single-page JavaScript game: find the hidden Super Smash Bros characters on the crowded image.
- From The Odin Project

Features
- Click to mark found characters; tracks time.
- High scores saved to Firebase.
- Magnifying glass toggle for a closer look.

Tech
- Vanilla JavaScript, HTML, CSS

Play
- Start a level, scan the image, click each hidden character to mark them found.
- Once you find all 3, try and top your best time!
